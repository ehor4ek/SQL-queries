# SQL-задачи на тестовых базах данных

В этом репозитории собраны решения задач по SQL на четырёх наборах данных:  
- **Vehicles** – автомобили, мотоциклы, велосипеды  
- **Races** – гоночные классы, автомобили, гонки и результаты  
- **Hotels** – отели, номера, клиенты и бронирования  
- **Organization** – работники, проекты, задачи

Для каждой темы приведены скрипты создания таблиц, наполнения тестовыми данными, описание задач и сами запросы‑решения.

## Структура репозитория
- Есть директории по названимя БД: vehicles, races, hotels, organizations
- В каждой директории файл tasks.md с задачами
- В каждой директории скрипт создания таблиц create_tables.sql
- В каждой директории скрипт наполнения данными insert_data.sql
- В каждой директории SQL-скрипты по задачам: task1.sql, task2.sql...



## Как использовать

1. **СУБД**  
   Все запросы написаны и протестированы на **PostgreSQL 17**.  
   В файлах `create_tables.sql` и `insert_data.sql` есть блоки для MySQL, но основные решения ориентированы на PostgreSQL.

2. **Создайте базу данных**  
   Выполните соответствующую часть `create_tables.sql` в вашей среде PostgreSQL.

3. **Заполните таблицы**  
   Выполните скрипт `insert_data.sql`.

4. **Запустите решение**  
   Откройте нужный `taskN.sql` и выполните запрос.

## Требования
- PostgreSQL 17 (или более поздняя версия)
